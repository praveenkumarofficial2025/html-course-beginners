Common Attributes

src → URL of the page or content

width, height → Size of the iframe

frameborder → Border of the frame (0 = no border)

allowfullscreen → Allows embedded video to go full screen

loading="lazy" → Delays loading until iframe is visible (improves performance)

Security Note

Some websites block iframe embedding for security using headers like X-Frame-Options.

Always verify the content you embed to avoid security issues.

Tips

✔ Use <iframe> to embed maps, videos, forms, and external tools.

✔ Set appropriate width and height for better layout control.

✔ Always include frameborder="0" and allowfullscreen for videos when needed.
